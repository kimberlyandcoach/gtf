Microsoft Teams Self‑Paced Challenge (GTCF) — “The Grant Cycle Surprise”

Purpose: A self‑paced challenge where students practice a wide range of Microsoft Teams skills (chat, channels, meetings, files, tabs, notifications, version history, planning, Loop, Whiteboard, and Copilot-style summaries).

Time:** 60–90 minutes (self‑paced)

Format:** Individual (recommended) or pairs

Deliverables:** Concrete artifacts in Teams (posts, files, tasks, meeting notes) + a short final wrap‑up message.

Storyline

It’s Grant Cycle Launch Day at the Greater Toledo Community Foundation.

At 9:30 AM, you learn that a local reporter wants a 12:00 PM press briefing about the new grant timeline, and at the same time, your programs team discovers a missing required document in the application packet.

Your mission: use Teams to quickly coordinate, produce a clean “single source of truth,” and deliver a tight briefing plan—without creating channel sprawl.

Setup (2–3 minutes)

Create (or use) a Team named “GTCF | Grants & Programs” and (if allowed) add these channels:

  • 01-Announcements
  • 02-Grant Cycle Launch
  • 03-Press Briefing
  • Resources-Templates

If you can’t create Teams/channels, use an existing Team and create the same structure in channel names or a folder layout. (The learning goal is planning + organization.)

Challenge Rules

  1. Work independently (no trainer help unless you’re stuck for >5 minutes).
  2. Every task must produce a visible artifact (message, file, task, tab, whiteboard, etc.).
  3. Use the “Evidence Checklist” at the end to self‑verify.

Challenge Tasks (Self‑Paced)

Part 1 — Communication Under Pressure (15 minutes)

Task 1: Chat like a pro

In a 1:1 chat with your “Manager” (real or role-play), send a message that includes:

  • an @mention
  • formatting (bold a deadline)
  • a reaction to their message
  • a shared file or link (can be a blank doc link)
  • then search chat for “briefing” or “grant” to locate a detail you posted earlier

Artifact: 1 chat message + proof you used search (e.g., quote the found line in a follow‑up).

Task 2: Decide Chat vs Channel

Post a short note explaining what belongs in:

  • Chat (quick coordination / sensitive)
  • Channel (shared visibility / ongoing work)

Artifact: A 2–3 sentence message posted to 02-Grant Cycle Launch.

Part 2 — Channel Collaboration & Tabs (10 minutes)

Task 3: Post vs Reply (thread discipline)

In 02-Grant Cycle Launch, create a new post titled:

“Grant Cycle Launch – Issue Log”

Then add at least two replies in the same thread:

  1. “Missing document identified…”
  2. “Owner assigned + ETA…”

Artifact: 1 post + 2 replies in a single thread.

Task 4: Add a useful tab

Add a tab to 03-Press Briefing:

  • File tab for “Briefing Agenda,” or
  • Planner tab (if you prefer to do planning first)

Artifact: 1 tab added.

Part 3 — Meetings: Briefing Prep (15 minutes)

Task 5: Schedule and run a quick meeting

Schedule a Teams meeting called:

“Press Briefing Prep – 15 min”

During the meeting:

  • Use meeting controls (mute/unmute + one more: reaction/raise hand/captions)
  • Post a message in meeting chat
  • Share your screen for 30–60 seconds showing a draft agenda or checklist

Artifact: Meeting appears on calendar + meeting chat message + screen share completed.

Part 4 — Files & Content Hub (20 minutes)

Task 6: Create the “single source of truth”

In Resources-Templates, create a folder:

“Grant Cycle Launch Pack”

Add two files:

  1. Grant-Cycle-Timeline.docx (or .pptx)
  2. Press-Briefing-Agenda.docx

Then:

  • Open a file and edit it in Teams/desktop
  • Demonstrate co-authoring (pair if possible; if solo, open in browser + desktop, or edit then show version changes)

Artifact: Folder + two files created and edited.

Task 7: Version history + restore

In one of the documents:

  • Make a deliberate change (delete a paragraph)
  • Use Version history to view prior versions and restore the earlier version

Artifact: A restored version (or screenshot/description if restore isn’t allowed).

Part 5 — Planner + My Tasks/My Day (10 minutes)

Task 8: Build an action plan

Add Planner to 03-Press Briefing (tab) and create tasks:

  • Draft briefing agenda (Owner: you)
  • Confirm grant timeline dates (Owner: Programs)
  • Update website copy (Owner: Comms)
  • Prepare Q&A sheet (Owner: you)

Then open My Tasks / My Day and confirm at least one task is visible.

Artifact: Planner board with tasks + evidence of task in My Day.

Part 6 — Loop + Whiteboard + Copilot-style Summary (10 minutes)

Task 9: Loop component

In a channel post (or chat), insert a Loop checklist titled:

“Press Briefing Readiness Checklist”

Add 5 checkboxes (agenda, spokesperson, timeline, Q&A, publish link).

Artifact: Loop checklist created and edited (by you and a partner, if available).

Task 10: Whiteboard

Create a Whiteboard with three areas:

  • Key Messages
  • Likely Questions
  • Who Answers What

Artifact: Whiteboard saved/shared.

Task 11: Copilot-style summary

If Copilot is available, generate a summary of your Issue Log thread or meeting chat:

  • key decisions
  • action items + owners

If Copilot isn’t available, write a 5‑bullet manual summary in the channel.

Artifact: Summary message posted to 03-Press Briefing.

Part 7 — Personal productivity controls (5 minutes)

Task 12: Pin, mute, and notifications

  • Pin your most important chat or channel
  • Mute a low‑priority chat
  • Adjust channel notifications for 03-Press Briefing so you don’t miss updates

Artifact: Show pinned item + describe the notification change in one sentence.

Evidence Checklist (Students self‑grade)

Check each box when complete:

Step‑by‑Step Solution Document (Instructor Key)

Use this section as the answer key. It includes click‑paths, expected outputs, and quick troubleshooting notes.

Solution — Setup

  1. Create/choose Team
    • Teams app → Teams → Join or create team → Create team (if allowed).
    • Name: GTCF | Grants & Programs.
  2. Create channels (if allowed)
    • Team name → … (More options) → Add channel.
    • Create: 01-Announcements, 02-Grant Cycle Launch, 03-Press Briefing, Resources-Templates.

Success check: Team visible in Teams list; channels visible under the Team.

Solution — Part 1 (Chat & Chat vs Channel)

Task 1 solution (chat artifacts)

  1. Go to Chat → open 1:1 with Manager.
  2. Type a message like:
    • “@Manager Press briefing plan due by 11:00 AM. I’m drafting the agenda here: [link].”
  3. Apply formatting:
    • Highlight text → use B for bold.
  4. Use a reaction:
    • Hover a message → React (thumbs up/heart).
  5. Share a file/link:
    • In chat compose box → Attach (paperclip) → upload or share OneDrive link.
  6. Search chat:
    • Use the Search bar at top → type “briefing” → open result.

Expected artifact: A message containing @mention + bold + link/file; a reaction applied; a follow-up quoting the searched line.

Troubleshooting: If search results don’t show, ensure you’re searching from the top bar (not inside the message box).

Task 2 solution (chat vs channel)

  1. Open 02-Grant Cycle Launch.
  2. New conversation → post:
    • “Use channel for shared status + files; use chat for quick 1:1 coordination or sensitive info.”

Solution — Part 2 (Post vs Reply, Tabs)

Task 3 solution (thread discipline)

  1. In 02-Grant Cycle Launch → New conversation.
  2. Title: Grant Cycle Launch – Issue Log.
  3. Post the issue.
  4. Click Reply under your post and add two replies (not new posts).

Success check: All updates are nested under one thread.

Task 4 solution (add tab)

  1. Open 03-Press Briefing.
  2. Click + next to tabs.
  3. Choose File (or Planner).
  4. Select existing file or create a new one.

Success check: New tab appears at the top of the channel.

Solution — Part 3 (Meeting)

Task 5 solution (schedule + run)

  1. Schedule
    • Teams → Calendar → New meeting.
    • Title: Press Briefing Prep – 15 min.
    • Invite Manager/Colleague → Send.
  2. Join the meeting.
  3. Use controls:
    • Toggle Mute.
    • Use Raise hand or Reactions.
  4. Meeting chat:
    • Post a short status (“Agenda draft in Resources‑Templates folder.”).
  5. Screen share:
    • Share → Screen/Window → show agenda doc/checklist for 30–60 sec.

Success check: Meeting shows in calendar; meeting chat contains your message; participants saw screen share.

Solution — Part 4 (Files + Content Hub + Version History)

Task 6 solution (content hub)

  1. Open Resources-Templates channel.
  2. Go to Files tab.
  3. New → Folder → name: Grant Cycle Launch Pack.
  4. Create files:
    • New → Word document → name: Grant-Cycle-Timeline.
    • New → Word document → name: Press-Briefing-Agenda.
  5. Edit:
    • Open a doc → type content.
  6. Co-author:
    • If paired: both open doc at same time and type in different sections.
    • If solo: open in Teams and browser; make changes; confirm autosave/version updates.

Success check: Folder and files exist; file shows edits.

Task 7 solution (version history + restore)

  1. In a document library (Files tab), locate the file.
  2. Click … next to file → Version history.
  3. Select a prior version → Restore.

Success check: Deleted content returns; version history shows the restore event.

Solution — Part 5 (Planner + My Day)

Task 8 solution (Planner tasks)

  1. Go to 03-Press Briefing channel.
  2. Add tab: + → Planner.
  3. Create a plan (or use existing).
  4. Add tasks:
    • Draft briefing agenda (assign to you)
    • Confirm grant timeline dates (assign Programs)
    • Update website copy (assign Comms)
    • Prepare Q&A sheet (assign you)
  5. Verify in Tasks/Planner app:
    • Open Tasks by Planner and To Do → My Day (or My Tasks) → confirm at least one assigned task appears.

Troubleshooting: If tasks don’t appear in My Day, ensure the task is assigned to you and due today, then add to My Day.

Solution — Part 6 (Loop + Whiteboard + Copilot summary)

Task 9 solution (Loop checklist)

  1. In a channel post or chat compose box:
  2. Choose Loop components → Checklist.
  3. Title: Press Briefing Readiness Checklist.
  4. Add five items; if paired, have partner check/uncheck an item.

Task 10 solution (Whiteboard)

  1. In Teams, open Whiteboard:
    • From a meeting: Share → Microsoft Whiteboard, or
    • From app launcher (depending on tenant settings).
  2. Create three labeled areas and add sticky notes.
  3. Save/share link in 03-Press Briefing channel.

Task 11 solution (Copilot or manual)

  • If Copilot available: In the meeting chat or channel thread, use Copilot to summarize and list action items.
  • If Copilot unavailable: Post a manual summary with:
    1. Key decisions
    2. Action items + owners
    3. Deadlines
    4. Link to the “Grant Cycle Launch Pack” folder
    5. Next check‑in time

Solution — Part 7 (Pin/Mute/Notifications)

Task 12 solution

  1. Pin
    • For chat: right‑click chat → Pin.
    • For channel: open channel → … → Pin.
  2. Mute
    • Right‑click low priority chat → Mute.
  3. Notifications
    • Channel → … → Channel notifications → set to “All activity” or “Mentions” as desired.

Success check: Pinned items appear at top; muted chat indicates muted; channel notification setting updated.

Instructor Scoring (quick rubric)

  • Complete (✅): Artifact exists and matches task.
  • Partial (➖): Attempted but missing required element (e.g., no restore done).
  • Redo (🔁): Wrong place (chat instead of channel) or no evidence.

Optional Extension (if students finish early)

  • Add a naming conventions note in 01-Announcements explaining why the channel numbering reduces sprawl.
  • Create a short “Q&A” doc in the Launch Pack and link it as a tab in 03-Press Briefing.
